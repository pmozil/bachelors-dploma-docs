\chapter{Detailed FPGA Synthesis Area Reports}
\label{app:detailed_tables}

This appendix provides the per-instance cell counts produced by Vivado
synthesis for each of the three CXU configurations evaluated in
Chapter~\ref{ch:experiments_results}.

All figures are for the Digilent Arty S7-50 (Xilinx Spartan-7 XC7S50) target,
synthesised with Vivado in out-of-context mode as part of the LiteX SoC build flow.
The \emph{Cells} column reports the number of primitive logic primitives
(LUTs, flip-flops, carry chains, etc.) assigned to each hierarchical module
after synthesis.
